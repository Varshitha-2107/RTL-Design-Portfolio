# Streaming Sensor Smoother

## Overview
This project implements a streaming sensor smoother in Verilog HDL to reduce noise in real-time sensor data. The design processes incoming samples continuously and produces smoother output values suitable for digital signal processing and embedded applications.

## Features
- Real-time streaming data processing
- Synthesizable Verilog RTL
- Low-latency operation
- Parameterizable design (if applicable)
- Verified using simulation

## Applications
- IoT sensor data processing
- Industrial monitoring systems
- Embedded systems
- Digital signal preprocessing

## Files
- `rtl/streaming_sensor_smoother.v` – RTL implementation
- `tb/streaming_sensor_smoother_tb.v` – Testbench
- `waveforms/sim.png` – Simulation waveform

## Tools Used
- Verilog HDL
- Icarus Verilog
- GTKWave

## Simulation
The design was simulated to verify correct smoothing of streaming sensor data under different input conditions.
