# RTL Design Portfolio

## Overview

This repository serves as an index of my RTL design projects implemented in Verilog HDL. Each project focuses on a core digital design concept and includes synthesizable RTL, a dedicated testbench, simulation results, and documentation.

The goal of this portfolio is to progressively build expertise in RTL design, FPGA development, and Digital VLSI through hands-on implementation of industry-relevant hardware blocks.

---

## RTL Projects

| Project | Description |
|---------|-------------|
| Streaming Sensor Smoother | Moving average filter for real-time sensor data smoothing. |
| 8-bit ALU | Arithmetic Logic Unit supporting arithmetic and logical operations. |
| Parameterized Round Robin Arbiter | Fair arbitration using rotating priority scheduling. |
| Parameterized Synchronous FIFO | Configurable FIFO with full and empty flag generation. |
| ... | Additional projects will be added as the portfolio grows. |

## Development Flow

Each project follows a consistent workflow:

1. RTL Design
2. Functional Verification using a Verilog Testbench
3. Simulation using GTKWave
4. Documentation with Block Diagrams
5. Version Control using Git and GitHub
